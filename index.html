FizzBuzz

Consider the following problem:

Write a program that takes in all integers from 1 to 100 and returns 'Fizz' if it is divisble by 3, 'Buzz' if it is divisible by 5, 'FizzBuzz' if it is divisible by both and finally just prints the number otherwise.

This is a classic problem in programming and below is my solution in Python

for i in range(1,101):
    if i%3==0 and i%5==0:
        print('FizzBuzz', end = ' ')
    elif i%3==0:
        print('Fizz', end = ' ')
    elif i%5==0:
        print('Buzz', end = ' ')
    else:
        print(i, end = ' ')
